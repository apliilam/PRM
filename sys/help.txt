Property Release Maker (PRM) - конструктор для заполнения универсальной формы Property Release.


Сокращения:
 - Property Release - универсальная форма релиза для подачи творческих работ (Файлов) на микростоки.
 - Файл - творческая работа в области дизайна/иллюстрации.
 - Референс - скан/фото/скетч исходного изображения, на основе которого создан Файл, в формате jpg, png, bmp.
 - Батч - несколько файлов на основе одного Референса.

 - ЛКМ - левая кнопка мыши
 - ПКМ - правая копка мыши


Возможности: 
 - создание одного Property Release для Батча;
 - автоматическое создание Property Release для каждого файла в Батче;
 - объеденение нескольких Референсов в одном Property Release; 
 - возможность редактирования Референсов (поворот, масштабирование, обрезка, расположение);
 - автоматическое заполнение даты из Файлов и Референсов.

  

Инструкция:  
  1) Загрузить файл(ы) в список "File(s) list":
    - загрузка: либо нажатием кнопки "Open File(s)", либо перетаскиванием файлов из папки в список "File(s) list";
    - нажатием кнопки "Add Custom" можно добавить своё название (Enter-применить);
    - сортировка списка: перетаскиванием Файлов в списке;
    - добавить можно до 40 файлов;
  2) Загрузить и скомпоновать Референсы:
    - загрузка: либо двойной клик ЛКМ, либо перетаскиванием файлов из папки в область работы с Референсами;
    - ПКМ: меню для редактирования Референса (обрезка, поворот, и т.д.);
    - удерживанием ЛКМ: позиционирование;
    - Скролинг мышью: масштабирование;
  3) Заполнить(проверить) поля формы Property Release (Enter-применить):
    - если авторские права на Референс принадлежат фотографу, то ставим галку на "I am Owner and Photographer" (поля для Owner заполняются автоматически);
    - нажимая на "Signature" (кликабельно), подгружаем файл подписи (*.png с прозрачным фоном, приблизительный размер 160х60px);
    - поле "Date" для Photographer/Owner заполняется автоматически из последнего добавленного Файла/Референса соответственно;
    - ПКМ: возможность заполнить поле "Date" для Photographer/Owner из выбранного Файла/Референса;
    - поле "Name/Description" заполняется автоматически (либо из "Title" метаданных файла, либо имя файла);
  4) Нажать кнопку "create Property Release":
    - сохраненный Property Release будет иметь имя "выделенного Файла"PR.jpg;
    - галка "Cut PR-file up to 30": обрезание имени файла Property Release до 30 символов (необходимо для StockSubmitter);
    - галка "Copy PR-file name": копирование имени последнего созданного файла Property Release в буфер обмена;
    - галка "One Release per File": автоматическое создание отдельного Property Release на каждый Файл в списке;
    - галка "Put in the same directory": Property Release сохраняется в папку выделенного файла;
  


Протестировано на Win7(х64) для: Istockphoto, Shutterstock, Fotolia, Dreamstime, Bigstockphoto, 123RF, Depositphoto
